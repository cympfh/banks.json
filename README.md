# banks.json
Banks in Japan

## Data source

- [銀行コード検索](http://ykaku.com/ginkokensaku/index.php)
- [github:zengin-code/source-data](https://github.com/zengin-code/source-data)

## What is different from zengin-code?

Add new fields: "type" and "suffix"

| field | values |
|:------|:-------|
| type  | ginko  |
| | sinkin (信用金庫) |
| | sinkumi (信用組合) |
| | rokin (労金) |
| | nokyo (農協) |
| | gyokyo (漁協) |
| | shoko (商工中金) |
| | yucho (ゆうちょ) |
| suffix | "銀行" or empty |
